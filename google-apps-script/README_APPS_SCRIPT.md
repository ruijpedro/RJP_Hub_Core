# RJP Hub Core V3 - Apps Script

1. Abre o teu projeto Apps Script do RJP Hub.
2. Substitui o `Code.gs` por este ficheiro.
3. Guarda.
4. Implementar → Gerir implementações → Editar → Nova versão → Implementar.
5. Usa o URL `/exec` na app.

Ações principais:
- `ping`
- `syncAll`
- `calendar.list`
- `calendar.add`
- `calendar.sync`
- `study.import`
- `study.publish`
- `swim.import`
- `swim.publish`
- `drive.ensureStructure`
- `sheets.get`, `sheets.save`, `sheets.append`
