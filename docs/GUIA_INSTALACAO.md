# Guia rápido de instalação

## GitHub

1. Cria o repositório `RJP_Hub_Core`.
2. Envia todos os ficheiros e pastas do ZIP para a raiz.
3. Vai a Actions.
4. Executa `Build WebApp` e `Build Android APK`.

## Apps Script

1. Abre https://script.google.com.
2. Abre o projeto RJP Hub.
3. Cola o conteúdo de `google-apps-script/Code.gs`.
4. Guarda.
5. Implementar → Gerir implementações → Editar → Nova versão → Implementar.
6. Copia o URL `/exec`.
7. Cola na app RJP Hub Core.
